Using PDFlib TET with Python
============================

Python 3
--------
Python 3.3, 3.4 and 3.5 are supported.

Please use the Python 3.3 binding for Python 3.4 and 3.5.

Exception:
For Windows a extra Python 3.5 binding is available.


Python 2.5/2.6/2.7 (on Linux/Unix/OS X)
---------------------------------------
Python 2.5, 2.6 and 2.7 use PYTHON_API_VERSION 1013, so they can
use the same PDFlib TET binaries.

Please use the Python 2.5 binding for Python 2.6/2.7.

Exception:
On OS X 10.7/10.8/10.9 please use the Python 2.6 binding for Python 2.7.
